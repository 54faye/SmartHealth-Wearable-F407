# STM32F407 智能手表 - 硬件引脚连接说明

## 开发板基本信息
- **MCU**: STM32F407VET6
- **晶振**: HSE 25MHz
- **系统时钟**: 168MHz
- **标准库**: STM32F4xx StdPeriph

---

## 模块引脚分配总览

| 模块 | 功能 | F407引脚 | 说明 |
|------|------|----------|------|
| OLED | SCL | PB14 | 软件I2C时钟 |
| OLED | SDA | PB15 | 软件I2C数据 |
| MPU6050 | SCL | PB12 | 软件I2C时钟 |
| MPU6050 | SDA | PB13 | 软件I2C数据 |
| KEY1 | 按键1 | PA0 | 高电平有效，下拉 |
| KEY2 | 按键2 | PA1 | 低电平有效，上拉 |
| KEY3 | 按键3 | PA4 | 低电平有效，上拉 |
| LED | 板载绿灯 | PC5 | 低电平点亮 |
| USART1 | TX | PA9 | 串口发送 |
| USART1 | RX | PA10 | 串口接收 |
| RTC | LSE | PC14/PC15 | 32.768kHz外部晶振 |

## 详细接线说明

### 1. OLED显示屏（0.96寸 SSD1306，I2C接口）

```
OLED模块        F407开发板
------          ------
VCC    ------>  3.3V
GND    ------>  GND
SCL    ------>  PB14
SDA    ------>  PB15
```
- 使用软件I2C（GPIO模拟），不占用硬件I2C资源
- OLED驱动位于 `bsp_oled.c`

### 2. MPU6050 六轴传感器（I2C接口）

```
MPU6050模块     F407开发板
------          ------
VCC    ------>  3.3V
GND    ------>  GND
SCL    ------>  PB12
SDA    ------>  PB13
AD0    ------>  GND（地址0xD0）
```
- 使用软件I2C（GPIO模拟），I2C地址 0xD0
- MPU6050驱动位于 `bsp_mpu6050.c`

### 3. 按键（3个）

```
按键模块        F407开发板    电气特性
------          ----------    --------
KEY1   ------>  PA0          高电平有效，配置下拉输入
KEY2   ------>  PA1          低电平有效，配置上拉输入
KEY3   ------>  PA4          低电平有效，配置上拉输入
```
- 使用TIM2定时器中断轮询（10ms周期），软件消抖（20ms）
- 按键驱动位于 `bsp_key.c`

### 4. LED指示灯

```
LED             F407开发板
---             ----------
LED1  ------>   PC5（板载绿色LED）
```
- 低电平点亮
- LED驱动位于 `bsp_led.c`

### 5. USART1 串口调试

```
F407开发板       USB转TTL模块
----------       ------------
PA9 (TX)  ------>  RXD
PA10 (RX) ------>  TXD
GND       ------>  GND
```
- 波特率：115200
- 用于printf重定向输出调试信息
- 串口驱动位于 `bsp_usart.c`

### 6. RTC 实时时钟

```
F407开发板
----------
PC14/PC15  ------>  32.768kHz LSE晶振（板载）
```
- 使用外部低速晶振LSE
- 电池备份域保持时间（需要纽扣电池）
- RTC驱动位于 `bsp_rtc.c`

## 注意事项

1. **PA0引脚冲突**：KEY1和ADC都映射到PA0，目前KEY1优先。如需使用ADC功能，需更换到其他ADC通道引脚。
2. **软件I2C**：OLED和MPU6050均使用软件I2C（GPIO模拟），可以使用任意GPIO引脚，不依赖硬件I2C外设。
3. **FreeRTOS**：使用ARM_CM4F移植层，堆大小32KB，系统节拍1kHz。
